# M5-ADV-Basketball-game
just a simple game for the cardputer adv in its first version 0.1.
this game has flashinglights so warning if your sensitive to flashing lights.
to get the firmware on the device simply open the arduino ide download and set the board to the m5 cardputer adv insure your cardputer is the advanced model then copy the code from main.cpp and paste it into your file from here plug in your cardputer adv and set the port to the port your device is connected to and download the library's and dependencies found in main.cpp then click the burn button and wait for the firmware to load.
